Liveramp Segment Creator

Enter a buzz key, upload a Liveramp taxonomy file, and press "Create Segments" to create a segment for each row with a delta of "TRUE".

Items to Note:

1. This tool uses your browser's cookies to authenticate. The segments will be created under the account ID you're currently logged into, of the buzz key you enter below. Ensure you're logged into the proper account beforehand.
2. You may specify which column number(s) to use as the segment's name below as a comma-delimited list. Multiple values will be joined with a space.
3. It is assumed that the "ID" will be in column 1, "description" in column 6, and "delta" in column 11. Each segment is assigned a TTL of 90 days.

Column(s) To Use as Segment Name - enter column numbers as a comma-delimited list. Multiple values will be joined with a space.
